# BasaiSaraiProject

## Overview

**BasaiSarai** is a mobile application designed to simplify the room rental process in densely populated cities like Kathmandu. This user-friendly platform connects tenants with homeowners, facilitating direct communication and enhancing the overall rental experience. With features like location-based searches, secure payment options, and plans for future integrations, BasaiSarai aims to address the common challenges faced by individuals seeking rental accommodations.

## Features

- **Direct Chat with Room Owners:** Users can communicate directly with homeowners, fostering transparency and efficient negotiations.
- **Location-Based Search:** Easily find available rooms based on specific geographical preferences to ensure proximity to workplaces, schools, or other points of interest.
- **Keyword-Wise Room Search:** Users can filter search results by price and other criteria to find accommodations that fit their budget.
- **Feedback System:** Tenants can leave feedback on their rental experiences, helping future users make informed decisions.
- **Future Enhancements:** Planned features include second-hand furniture sales and integration with local moving services.

## Technologies Used

- **Frontend:** Developed using [React Native](https://reactnative.dev), allowing for cross-platform compatibility on both Android and iOS devices.
- **Backend:** Powered by [Firebase](https://firebase.google.com) for seamless server-side logic and real-time data handling.
- **Database:** Utilizes [Firestore](https://firebase.google.com/docs/firestore) for storing user profiles, room listings, and more.
- **Authentication:** Implements [Clerk](https://clerk.dev) for secure user authentication and authorization, ensuring user privacy and data security.
- **Mapping:** Integrated with the [Google Maps API](https://developers.google.com/maps/documentation) for geocoding and displaying room locations through the Expo framework.
